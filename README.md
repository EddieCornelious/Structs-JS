A simple data structure library

## Included data structures

* Linked List
* Binary Search Tree
* Queue
* Stack
* Trie
* HashMap
* HashSet
* Array Methods
* Priority Queue
* Binary Heap
* Graph
* Set
* Map

## Getting Started

```
npm install js_data-collections
```
