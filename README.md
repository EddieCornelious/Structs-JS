# Js_Data-Collections

A simple data structure library

## Included data structures

* Linked List
* Binary Search Tree
* Red-Black Tree
* Queue
* Stack
* Trie
* HashMap
* HashSet
* Array Methods
* Priority Queue
* Binary Heap
* Graph
* HashMultiMap
* Set
* Map

## Getting Started

```
npm install js_data-collections
```
