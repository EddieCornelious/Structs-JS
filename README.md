## Included data structures &#128296;

* Linked List &#10004;
* Binary Search Tree &#10004;
* Queue &#10004;
* Stack &#10004;
* Trie &#10004;
* HashMap &#10004;
* HashSet &#10004;
* Array Methods &#10004;
* Priority Queue &#10004;
* Binary Heap &#10004;
* Graph &#10004;
* Set &#10004;
* Red-Black Tree &#10004;
* MultiMap &#10004;
