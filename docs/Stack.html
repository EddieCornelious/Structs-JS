peek() → {*}

Reports but does not remove the staged element to be removed next
Example
stack.push(1);
stack.peek() // returns 1 but does not remove it
Returns:
The element staged to be removed next

*

pop() → {*}

Removes data from stack in a last in first out manner
Example

stack.pop(); // 3
Returns:
The reomved data

*
